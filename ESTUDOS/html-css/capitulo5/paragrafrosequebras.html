Paragráfos e quebras de linha

Esse é o exercício 002

Você pode escrever um paragráfo de qualquer jeito. basta colocar tudo no meio do par de tags < p> e < /p>

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag < br>

A tag 'br' gera uma quebra de linha ou quebrar o texto em algum lugar especifico significa break row.

Vamos adicionar simbolos especiais: ® © ™ € £ ¥ ¢ δ Δ ↑

Vamos adicionar alguns emojis: No site "emojipedia.org" tem os Codepoints para copiar e adiciona aqui da seguinte forma: 🖖 🤓

Para adicionar um comentário basta escreever dentro da tag <!-- <-> LESS THAN, >-> GRATER THAN -->
