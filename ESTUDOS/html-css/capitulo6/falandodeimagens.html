Formato de imagens

Para web, geralmente usamos formatos de imagens JPEG e PNG. Algumas poucas versões usamos o GIF, TIF, SVG. Mas basicamente os desenvolvedores se viram muito bem com o JPE e PNG

Imagens JPEG não suportam images com fundo transparente, somente no formato PNG.

O tamanho das imagens importa para um site?

Um tamanho ideal para usar em um site seria, não necessariamente, 1500 de largura e o proprio editor de imagens já faz a altura proporcional. A resolução pode ser 50,000 e a qualidade pode por 70%. O tamanho 650 também é ótimo para utilizar em sites, assim teremos imagens mais leves, não pesando tanto o site, isso é muito bom para a economia na hora de escolher o plano de hospedagem.

A tag img em HTML5

Para colocar imagens em HTML5 é só digitar 'img' e dar enter que o proprio VScode ja deixa tudo pronto. (ex: <img src="" alt="">) dentro das aspas de src=significa sourcec ou origem, nela colocamos o caminho da imagem e dentro das aspas alt=significa alternative, é onde colocamos o texto alternativo que explica melhor a imagem. Colocando o cursor do mouse dentro das aspas de src e apertando espaço ele já da o caminho com o nome dos arquivos existentes para adicionar, o texto alternativo serve para leitores de tela, acessibilidade para pessoas deficientes visuais, por isso é importante que se coloque um bom texto alternativo.
