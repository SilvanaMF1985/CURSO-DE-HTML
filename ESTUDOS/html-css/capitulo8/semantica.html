O que é semâtica?

substantivo feminino 1. LINGUÍSTICA estudo sincrônico ou diacrônico da significação como parte dos sistemas das línguas naturais. 2. LINGUÍSTICA num sistema linguístico, o componente do sentido das palavras e da interpretação das sentenças e dos enunciados.

HTML está focado na Semântica e semântica significa significado

Então colocar um texto no meio não é significado é forma, colocar uma fonte branca não é significado é forma e pra tudo que é forma agora estão reservados as CSS, as folhas de estilos, pois o html está focado no significado.
