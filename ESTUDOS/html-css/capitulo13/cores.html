Como criar um degrade

Para criar um degrade como o fudo dessal tela, precisamos fazer os seguintes passo

passo 1: criar uma folha de estilo, ou criar dentro do próprio arquivo ou criar um arquivo separado, o que é mais rrecomendado

Passo 2: Criar a tag Body e dentro dela utiliar o background-image e escolher a função interna linear-gradiente, dentro do parenteses do gradiente linear escolher a direção, depois as cores, veja o exemplo.

bory{background-image: linear-gradient(to right, white, blue);}

As direções são representadas de 4 formas

  1. to right -> para direita
  2. to left -> para a esquerda
  3. to botton -> para baixo
  4. to top -> para cima
