Aula 1 e 2 de iframe

IFRAME significa iframe inline

<iframe src="endereço do site" frameborder="0">texto</iframe>

*Alguns sites não permitem iframes

Por isso é bom colocar dentro da tag iframe a tag 'a' com o mesmo link do site para caso o dispositivo não seja compativel com o iframe

Por padrão a altura e largura do iframe é 300x150, porém é possível configurar elas dentro da tag iframe ou no style. Podemos usar o pareametro scolling com os parametros 'auto', 'yes', 'no'.
